# Real Estate Page

A simple real estate UI layout built using HTML, CSS, and JavaScript, optimized for performance with lazy loading images and responsive design.

## Features

- Lazy loading for all property images
- Responsive grid layout
- Clean UI for mobile and desktop

## How to Run

1. Clone the repository or download the zip.
2. Open `index.html` in any browser.
3. No server required.
